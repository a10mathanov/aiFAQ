# 📝 История изменений (Changelog)

## [1.1.0] - 2026-06-16

### Добавлено 🎉
- **Поддержка SOCKS5 прокси** для Telegram Bot API
  - Добавлена библиотека `aiohttp-socks`
  - Bot инициализируется через `ProxyConnector`
  - YandexGPT API также использует прокси
  
- **Новые файлы:**

- **Улучшения в `services/yandex_gpt.py`:**
  - Поддержка параметра `proxy_url` в конструкторе

  - Fallback на прямое соединение если прокси недоступен

  - YandexGPT сервис инициализируется с прокси

- **Улучшения в `main.py`:**
  - Асинхронная функция `create_bot_with_proxy()` для инициализации бота с прокси
  - Bot использует `AiohttpSession` с `ProxyConnector`
  - Логирование информации о подключении к прокси

## Версионирование

Проект следует [Semantic Versioning](https://semver.org/):
- **MAJOR** версия: несовместимые изменения API
- **MINOR** версия: новая функциональность, совместима с предыдущей
- **PATCH** версия: исправления ошибок

Формат: `MAJOR.MINOR.PATCH` (например, `1.1.0`)

## Миграция между версиями

### Обновление с 1.0.0 на 1.1.0

1. Обновите зависимости:
   ```bash
   pip install -r requirements.txt --upgrade
   ```

2. Новый файл `PROXY_SETUP.md` содержит информацию по конфигурации прокси

3. Если вы не используете прокси, бот будет работать как раньше

4. Если вы хотите использовать прокси:
   - Отредактируйте `PROXY_URL` в `main.py` и `handlers/user_handlers.py`
   - Или используйте переменные окружения (см. `PROXY_SETUP.md`)
